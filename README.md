### Run local with API üè°

- Clone repo

- Install dependencies:
```shell
  npm i
```

- Run API (on API folder):
```shell
  cd API/

  docker-compose build

  docker-compose up
```

- Run APP (on root folder):
```shell
  npm run start
```
